# 🛡️ Java Defence System

A basic Java-based simulation project demonstrating various military defense units such as tanks, helicopters, and submarines. Built for educational purposes to explore object-oriented programming, class relationships, and controller-based coordination in Java.

---

## 📁 Project Structure

- `DefenceSystem.java` – Main application launcher  
- `MainController.java` – Central logic/controller for the system  
- `DefenceUnit.java` – Base class for defense units  
- `Tank.java`, `Helicopter.java`, `Submarine.java` – Specialized units extending `DefenceUnit`  
- `DefenceUnitList.java` – Utility class for managing multiple units

---

## 🚀 How to Run

1. **Requirements**:
   - JDK 8 or later
   - Java IDE (NetBeans, IntelliJ, or VS Code) or terminal

2. **Steps**:
   - Clone the repository or download the ZIP
   - Open in your Java IDE or compile via terminal:
     ```bash
     javac -d bin src/defencesystem/*.java
     java -cp bin defencesystem.DefenceSystem
     ```

- Simple simulation of defense units

- Controller pattern for managing behavior
